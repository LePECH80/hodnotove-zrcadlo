# Podklady pro komunikaci — Hodnotové zrcadlo™

Vytvořeno z analýzy **222 reálných odpovědí ze 7 dokončených testovacích rozhovorů** (anonymně). Tohle není vymyšlený marketing — je to jazyk tvojí skutečné cílovky.

## Co kde najdeš

| Soubor | K čemu slouží |
|---|---|
| **01_messaging-kit.md** | ZÁKLAD všeho. Segmenty cílovky, bolesti, námitky, aha-momenty, hodnotové propozice, slovník značky. Sem chodíš pro slova. |
| **02_socialni-site.md** | Hotové hooky, storytelling posty, carousel osnovy, osobní posty. |
| **03_emailova-sekvence.md** | 5 mailů od bolesti k nákupu — pro mailing (Ecomail). |
| **04_vylepseni-landing.md** | Konkrétní „před → po" úpravy landing page podle toho, co rezonovalo. |

## Kam sáhnout pro nejrelevantnější informace (do budoucna)

1. **🥇 Reálné rozhovory testerů** (Supabase, tabulka `sessions`) — nejcennější zdroj. Autentický jazyk cílovky. Z nich je celá tato složka.
2. **🥈 Vygenerované reporty** (sloupec `report`) — hmatatelný výstup, proof.
3. **🥉 Systémový prompt** (`src/lib/systemPrompt.ts`) — metodika, odlišení, „jak to funguje".
4. **Zpětná vazba testerů** — odhaluje, co rezonuje a co brzdí.

## Doporučené pořadí práce
1. Přečíst **messaging kit** (01) — nasát jazyk.
2. Upravit **landing** podle 04 (priorita: sekce s reálnými reakcemi).
3. Nasadit **e-mailovou sekvenci** (03) do Ecomailu.
4. Průběžně publikovat **posty** (02).
